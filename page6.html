[Sep 08, 2013] PlanetHour v1.0 App released:

The 'grah hora' or 'planetary hour' has traditionally been used in electing favorable times to initiate activities in our daily life. It is among the oldest branch of astrology almost all over the world.

Sounak Software introduces PlanetHour app for your java supported mobile phones. This app uses a cool interface to show the planetary day and hour along with vedic astrology based special times viz. rahu kaal, gulika kaal and yamagandam. An hours list is also provided to to show all the planetary hours for a particular day.

[Sep 08, 2013] PlanetHour Mini app released:

PlanetHour Mini is the small size version of the PlanetHour app for smaller phones supporting java. This scaled version of Planethour software is for phones which doesn't support jar file size more than 300 kB. However this can be installed in high end phones (having screen resolution more than 240 x 320 pixels) supporting java, the screen resolution may suffer.
